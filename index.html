Image Search Abstraction Layer

by Kevin Madill (http://www.github.com/kmadill)


This simple API was created as part of Free Code Camp's curriculum. The exercise can be found here. If you wish to view the source code, you can find that here.

User stories:

  1. I can get the image URL, alt text and page urls for a set of images relating to a given search string.
  2. I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  3. I can get a list of the most recently submitted search strings.
